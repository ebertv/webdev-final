About the Spotify Wiki

The Spotify Wiki is a program meant to display information about any song found within Spotify's database. This website functions as a search engine and allows you to save results you have looked at. The credits page details more information about Spotify's API and the creators of this website.

How do I use it?

Type in the name of any song from Spotify and learn about its stats and information. By clicking the button on any of the song cards, you will be able to save the songs in your own personal library.
